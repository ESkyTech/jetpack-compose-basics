# Compose Showcase

A small Jetpack Compose sample app to practice **basic composables**, **state**, **Material 3**, **lists**, and **simple animations**.

## What’s inside

- **Onboarding screen** with a “Continue” action (state persisted with `rememberSaveable`)
- **LazyColumn** rendering a list of items
- **Expandable card item**
  - Toggle expand/collapse with an icon
  - Smooth size animation using `animateContentSize` + `spring`
- **Material 3** UI components (`Scaffold`, `Surface`, `Card`, `Button`, `IconButton`)
- Previews included (light/dark)

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
